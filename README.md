# Creating-VPNs


<h2>Description</h2>
This project consists of a creating a virtual machine via Azure and setting up a VPN with Proton VPN.
<br />


<h2>Technologies Used</h2>

- <b>Microsoft Azure</b> 
- <b>Proton VPN</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
